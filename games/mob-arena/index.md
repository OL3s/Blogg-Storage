A gladiator survival game built around fighting one wave, buying upgrades, and stepping back into the arena stronger for the next wave.

Mob Gladiator is about surviving structured arena fights, then using the reward window between waves to improve your build. The loop is simple: fight, earn, upgrade, and return to the arena with better tools for the next challenge.
